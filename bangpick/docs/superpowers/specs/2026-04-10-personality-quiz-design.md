# 趣味人格测试功能设计

## 概述

为 bangpick 新增趣味人格测试功能，包含两套测试：**决策人格测试**和**投资风格测试**。用户答完 8 道题后，由 AI 生成个性化幽默解读，并可生成分享卡片图片。核心目标：**提升趣味性和分享欲**。

## 方案选择

采用**方案 B：AI 生成个性化解读**。题目前端硬编码，计分本地完成，答完后调用 AI 生成个性化解读文案。每次结果不同，增加惊喜感和分享动力。

## 页面结构

### 入口

在**帮我选首页**（ChatPage）顶部新增醒目的测试入口卡片，不新增 BottomNav tab。

### 路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/quiz` | QuizPage | 测试选择页，展示两套测试卡片 |
| `/quiz/:type` | QuizPlayPage | 答题页，type = `decision` 或 `investor` |
| `/quiz/:type/result` | QuizResultPage | 结果页，人格类型 + AI 解读 + 分享 |

## 测试内容

### 决策人格测试（8 题）

**4 个维度**（每个维度 2 题）：

| 维度 | 两极 |
|------|------|
| 速度 | 快刀斩乱麻 ↔ 深思熟虑 |
| 依据 | 直觉感受 ↔ 数据分析 |
| 风险 | 冒险尝鲜 ↔ 稳妥保守 |
| 社交 | 独立决定 ↔ 参考他人 |

**6 种人格结果**：

| 人格 | 关键特征 |
|------|----------|
| 闪电侠 | 快 + 直觉 + 冒险 |
| 军师型 | 慢 + 数据 + 稳妥 |
| 赌神型 | 快 + 直觉 + 冒险 + 独立 |
| 民主型 | 慢 + 参考他人 |
| 完美主义者 | 慢 + 数据 + 稳妥 + 独立 |
| 随缘大师 | 快 + 直觉 + 稳妥 |

### 投资风格测试（8 题）

**4 个维度**（每个维度 2 题）：

| 维度 | 两极 |
|------|------|
| 周期 | 短线 ↔ 长线 |
| 分析 | 技术面 ↔ 基本面 |
| 心态 | 激进 ↔ 保守 |
| 操作 | 频繁交易 ↔ 买入持有 |

**6 种风格结果**：

| 风格 | 关键特征 |
|------|----------|
| 短线猎手 | 短线 + 技术 + 激进 + 频繁 |
| 巴菲特门徒 | 长线 + 基本面 + 保守 + 持有 |
| 量化极客 | 短线 + 技术 + 保守 |
| 佛系定投 | 长线 + 保守 + 持有 |
| 全能操盘手 | 长线 + 技术 + 激进 |
| 消息面玩家 | 短线 + 基本面 + 频繁 |

### 计分方式

每题 4 个选项，每个选项对 2 个维度加分（0-2 分）。答完 8 题后，按 4 个维度得分匹配最接近的人格/风格。

## AI 解读

### 流程

1. 答完 8 题后，将人格类型 + 各维度得分发给 `/api/chat`
2. System prompt："你是一个幽默毒舌的性格分析师"
3. AI 返回 JSON：
   - `title`：一句话标题（如"你就是那个点菜要翻 3 遍菜单的人"）
   - `description`：100 字个性化解读，幽默有梗
   - `tip`：一句实用建议
4. 等待期间显示趣味 loading（"正在窥探你的灵魂..."、"分析你的脑回路中..."）

### API 复用

复用现有 `/api/chat` 端点，不新增 API。System prompt 和 messages 在前端组装。

## 结果页布局

从上到下：

1. **人格/风格图标** — 大 emoji
2. **类型名称** — 大字居中（如"闪电侠"）
3. **维度雷达图** — 4 维度得分可视化（SVG 雷达图）
4. **AI 解读卡片** — 标题 + 描述 + 建议，带背景色的卡片样式
5. **操作栏**：
   - 「生成分享图片」— Canvas 渲染成图片，长按保存
   - 「再测一次」— 重新答题
   - 「换一套测试」— 跳到另一套

## 分享图片

Canvas 生成竖版卡片（适合朋友圈），内容：
- 人格名称 + emoji
- 雷达图
- AI 金句（title）
- 底部 bangpick 小水印

## 技术实现

### 新增文件

| 文件 | 职责 |
|------|------|
| `src/pages/QuizPage.jsx` | 测试选择页（两张测试卡片） |
| `src/pages/QuizPlayPage.jsx` | 答题页（进度条 + 单题卡片 + 滑动切换） |
| `src/pages/QuizResultPage.jsx` | 结果页（雷达图 + AI 解读 + 分享） |
| `src/lib/quizData.js` | 题目、选项、计分规则、人格定义 |
| `src/lib/quizLogic.js` | 计分函数 + 人格匹配算法 |
| `src/components/RadarChart.jsx` | SVG 雷达图组件 |
| `src/components/ShareCard.jsx` | Canvas 分享图片生成 |

### 改动文件

| 文件 | 改动 |
|------|------|
| `src/App.jsx` | 新增 3 条路由 |
| `src/pages/ChatPage.jsx` | 顶部新增测试入口卡片 |

### 数据存储

测试结果存 `localStorage`（key: `quiz_results`），保存最近一次每套测试的结果，刷新不丢失。

### 依赖

不新增任何外部依赖：
- 雷达图：原生 SVG
- 分享图片：原生 Canvas API

## 答题体验

- 每次显示一题，选完自动滑到下一题
- 顶部进度条显示答题进度
- 支持点击返回修改上一题
- 答完最后一题自动提交

## 错误处理

- AI 解读请求失败时，降级显示预设的人格描述文案（每个人格类型预写一段）
- 网络超时提示"AI 正在思考，请稍等"，允许重试
